# ✅ SpendWise AI

AI-powered personal finance analytics system.

## Features
- Expense categorization using ML
- Anomaly detection
- Subscription tracking
- AI-based financial advice
- Streamlit dashboard
- Model accuracy (~90%)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- FastAPI (optional)

## How to Run

pip install -r requirements.txt  
streamlit run dashboard/app.py
